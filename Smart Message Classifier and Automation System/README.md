# Smart Message Classifier & Automation System

## Overview
This project is an NLP-based message classification system that categorizes messages into spam, work, or personal and triggers automated actions based on the classification.

## Features
- Text classification using Machine Learning
- Real-time message prediction
- Automated actions (spam filtering, notifications)
- Lightweight and fast execution

## Tools
- Python
- Scikit-learn
- NLP (CountVectorizer, Naive Bayes)

## Output
- Classifies messages into:
  - Spam
  - Work
  - Personal
- Triggers automation actions based on category

## Use Cases
- Email filtering systems
- Chatbot automation
- Notification systems

## Future Enhancements
- Use advanced NLP models (BERT, LSTM)
- Integrate with real messaging platforms
- Add GUI using Flask or Streamlit
